В папке Examples собраны примеры отчетов для рабочих тегов.

Примеры для базы данных testDB.db:
 - example.group.qreport - пример работы с группировкой
 - example.student.qreport - пример вывода списка студентов из базы данных
 - example.multy_group.qreport - пример работы с несколькими группировками одновременно

***
Сборка src была сделана на Ubuntu в Qt.
